# 编译原理课程设计报告

## 一、课程设计目的

本课程设计旨在实现一个完整的编译器前端系统，包括词法分析器、语法分析器和语义分析器，深入理解编译原理的核心概念和实现技术。

### 1.1 设计目标

- 实现词法分析器：对源程序进行扫描，识别并分类各类单词符号
- 实现语法分析器：基于递归下降分析法，验证程序的语法正确性
- 实现语义分析器：构建符号表，生成四元式中间代码

## 三、词法分析器设计与实现

### 3.1 设计思想

词法分析器负责将源程序字符流转换为单词符号序列。采用状态机思想，通过逐个字符扫描，根据字符类型进入不同的处理分支。

### 3.2 核心数据结构

#### 3.2.1 Token类型定义

```cpp
enum class TokenType
{
    KEYWORD,        // 关键字：if, else, while, return, int, float
    IDENTIFIED,     // 标识符
    NUMBER,         // 数字
    STRING,         // 字符串
    OPERATOR,       // 运算符：+, -, *, /, =, <, >, ==, !=, <=, >=, ++, --
    DELIMITED,      // 分隔符：;, ,, (, ), {, }, [, ]
    END             // 文件结束标记
};
```

#### 3.2.2 Token结构

```cpp
struct Token
{
    TokenType type;      // 单词类型
    string lexeme;       // 单词值
    int line, column;    // 位置信息
    int int_value;       // 整数值
    double double_value; // 浮点数值
    string string_value; // 字符串值
};
```

#### 3.2.3 符号表结构

```cpp
struct SymbolEntry {
    string name;       // 符号名称
    TokenType type;    // 符号类型
    int scope;         // 作用域层级
    int line;          // 首次出现行号
    int col;           // 首次出现列号
    bool isArray;      // 是否为数组
    int arraySize;     // 数组大小
};
```

### 3.3 主要功能实现

#### 3.3.1 关键字识别

#### 3.3.3 扫描流程

```cpp
Token scan()
{
    // 1. 跳过空白字符
    while (isspace(peek)) peek = nextChar();
    
    // 2. 根据首字符类型分配处理函数
    if (isdigit(peek))      return getNumber();
    if (isalpha(peek))      return getAlpha();
    if (isDelimiter(peek))  return getDelimited();
    if (isOperator(peek))   return getOperator();
    
    // 3. 返回结束标记
    return Token(TokenType::END);
}
```

## 四、语法分析器设计与实现

### 4.1 设计思想

采用递归下降分析法，为每个非终结符编写一个递归函数。语法分析器基于LL(1)文法，通过向前看一个token来决定使用哪个产生式。

### 4.2 文法定义

#### 4.2.1 扩展巴克斯-瑙尔范式(EBNF)

```
stmts → stmt rest0
rest0 → stmt rest0 | ε

stmt → loc = expr ;
     | if ( bool ) stmt [ else stmt ]
     | while ( bool ) stmt

loc → id resta
resta → [ elist ] | ε

elist → expr rest1
rest1 → , expr rest1 | ε

bool → equality
equality → rel rest4
rest4 → == rel rest4 | != rel rest4 | ε

rel → expr rop_expr
rop_expr → < expr | <= expr | > expr | >= expr | ε

expr → term rest5
rest5 → + term rest5 | - term rest5 | ε

term → unary rest6
rest6 → * unary rest6 | / unary rest6 | ε

unary → factor
factor → ( expr ) | loc | num
```

### 4.3 递归下降分析器实现

#### 4.3.1 主要分析函数

| 函数 | 对应产生式 | 功能 |
|------|-----------|------|
| `stmts()` | stmts → stmt rest0 | 语句序列 |
| `stmt()` | stmt → loc=expr; \| if... \| while... | 单条语句 |
| `loc()` | loc → id resta | 左值（变量或数组元素） |
| `expr()` | expr → term rest5 | 表达式 |
| `term()` | term → unary rest6 | 项 |
| `factor()` | factor → (expr) \| loc \| num | 因子 |
| `bool_expr()` | bool → equality | 布尔表达式 |

#### 4.3.2 语句分析实现

```cpp
void stmt() {
    if (currentToken.type == IDENTIFIED) {
        loc();
        expect("=");
        expr();
        expect(";");
        printProduction("stmt -> loc = expr ;");
    }
    else if (currentToken.lexeme == "if") {
        expect("if");
        expect("(");
        bool_expr();
        expect(")");
        stmt();
        if (currentToken.lexeme == "else") {
            expect("else");
            stmt();
            printProduction("stmt -> if ( bool ) stmt else stmt");
        } else {
            printProduction("stmt -> if ( bool ) stmt");
        }
    }
    else if (currentToken.lexeme == "while") {
        expect("while");
        expect("(");
        bool_expr();
        expect(")");
        stmt();
        printProduction("stmt -> while ( bool ) stmt");
    }
}
```

## 五、语义分析器设计与实现

### 5.1 设计思想

语义分析器在语法分析的基础上，完成以下功能：
1. 构建符号表，记录变量的名称、类型、作用域等信息
2. 生成四元式中间代码，表示程序的语义动作

### 5.2 核心数据结构

#### 5.2.1 四元式结构

```cpp
struct Quadruple {
    string op;     // 操作符：+, -, *, /, =, j<, j<=, j>, j>=, jnz, j, []=, =[]
    string arg1;   // 操作数1
    string arg2;   // 操作数2
    string result; // 结果
    int label;     // 用于回填的标记
};
```

#### 5.2.2 语义分析辅助结构

```cpp
struct boolExprNode {
    int truelist;   // 真出口链表
    int falselist;  // 假出口链表
};

struct controlNode {
    int nextlist;   // 后续链表（用于控制流）
};

struct arrayNode {
    string inArray;  // 数组名
    int inNdim;      // 维度
    string inPlace;  // 偏移量
    string array;    // 结果数组
    string offset;   // 结果偏移
    string place;    // 结果位置
};
```

### 5.3 主要功能实现

#### 5.3.1 符号表管理

```cpp
void insertSymbol(const string &name, const TokenType &type, int line) {
    if (lookupSymbol(name) == nullptr) {
        scanner.table.insert(name, SymbolEntry(name, type, currentScope, line));
    }
}

SymbolEntry* lookupSymbol(const string &name) {
    return scanner.table.lookup(name);
}
```

#### 5.3.2 四元式生成

```cpp
void emit(const string &op, const string &arg1, const string &arg2, const string &result) {
    quadList.emit(op, arg1, arg2, result);
    outputFile << "(" << op << ", " << arg1 << ", " << arg2 << ", " << result << ")" << endl;
}

string newtemp() {
    return "t" + to_string(tempCounter++);
}
```

#### 5.3.3 回填机制

```cpp
void backpatch(int list, int target) {
    while (list != -1) {
        int next = quadList[list].label;
        quadList[list].result = to_string(target);
        list = next;
    }
}

int merge(int list1, int list2) {
    if (list1 == -1) return list2;
    if (list2 == -1) return list1;
    quadList[list1].label = list2;
    return list1;
}
```

### 5.4 表达式翻译

#### 5.4.1 算术表达式

```cpp
string expr() {
    string termPlace = term();
    return rest5(termPlace);
}

string rest5(string inPlace) {
    if (currentToken.lexeme == "+") {
        currentToken = scanner.scan();
        string termPlace = term();
        string t = newtemp();
        emit("+", inPlace, termPlace, t);
        return rest5(t);
    }
    else if (currentToken.lexeme == "-") {
        currentToken = scanner.scan();
        string termPlace = term();
        string t = newtemp();
        emit("-", inPlace, termPlace, t);
        return rest5(t);
    }
    return inPlace;
}
```

#### 5.4.2 关系表达式

```cpp
boolExprNode rop_expr(string arg) {
    boolExprNode ret;
    if (currentToken.lexeme == "<") {
        currentToken = scanner.scan();
        string exprPlace = expr();
        ret.truelist = makelist(nextquad());
        ret.falselist = makelist(nextquad() + 1);
        emit("j<", arg, exprPlace, "-");
        emit("j", "-", "-", "-");
    }
    // ... 其他关系运算符
    return ret;
}
```

### 5.5 控制流语句翻译

#### 5.5.1 while循环

```cpp
controlNode stmt() {
    // ...
    else if (currentToken.lexeme == "while") {
        currentToken = scanner.scan();
        int m1 = nextquad();           // 循环开始位置
        expect("(");
        boolExprNode boolNode = bool_expr();
        expect(")");
        int m2 = nextquad();           // 循环体开始位置
        controlNode stmt1 = stmt();
        backpatch(stmt1.nextlist, m1); // 循环体结束跳回判断
        backpatch(boolNode.truelist, m2); // 条件为真进入循环体
        ret.nextlist = boolNode.falselist;
        emit("j", "-", "-", to_string(m1)); // 无条件跳回判断
    }
    // ...
}
```

#### 5.5.2 if-else语句

```cpp
else if (currentToken.lexeme == "if") {
    currentToken = scanner.scan();
    expect("(");
    boolExprNode boolNode = bool_expr();
    expect(")");
    int m1 = nextquad();
    controlNode stmt1 = stmt();
    if (currentToken.lexeme == "else") {
        currentToken = scanner.scan();
        int m2 = nextquad();
        backpatch(boolNode.truelist, m1);
        int temp = merge(boolNode.falselist, stmt1.nextlist);
        controlNode stmt2 = stmt();
        ret.nextlist = merge(stmt2.nextlist, temp);
        backpatch(boolNode.falselist, m2);
    }
}
```
